# ppp-style-clang-format

A `.clang-format` configuration file based on Bjarne Stroustrup's PPP (Programming: Principles and Practice Using C++) Style Guide.

## Usage

Place the `.clang-format` file into your target directory (usually the root directory).

## Reference

- [PPP Style Guide (PDF)](https://www.stroustrup.com/Programming/PPP-style.pdf)
